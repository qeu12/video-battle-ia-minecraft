## 📂 About video-battle-ia-minecraft

This project shows the results when four advanced AIs attempt to recreate Minecraft as a web game. The code comes from ChatGPT, Grok, Gemini, and Claude. Each AI’s output is stored separately in this repository.

You will find folders named after each AI with their generated code inside:

- `/claude`: Code generated by Anthropic Claude  
- `/gemini`: Code generated by Google Gemini  
- `/grok`: Code generated by xAI Grok  
- `/chatgpt`: Code generated by OpenAI ChatGPT  

The purpose is to explore and compare how the same game can be built differently by these AI models. This is an educational and experimental project.
